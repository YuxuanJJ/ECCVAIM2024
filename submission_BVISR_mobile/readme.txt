This project is build based on https://github.com/ckkelvinchan/BasicVSR_PlusPlus
Please follow the Installation to install the environment.

model_RTVSRx3.py is our model file
model_RTVSRx3.pth is the checkpoint

rtvsrx3_run.sh is an example to run the inference, where $fileName, $outputFile, $ffmpegpath need to be replaced.

The results obtained by our team are stored in the file named `vmafoutputx3`.
